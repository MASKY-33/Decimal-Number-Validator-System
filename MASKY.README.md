# Decimal Number Validator System
This system continuously asks the user for a number and checks whether the input is a valid decimal value.
It uses a try/except block to detect invalid input and provides clear feedback.

## Features
- Runs in a continuous loop until the user types "stop"
- Converts valid numeric input into a float
- Prints a success message when the input is a valid decimal number
- Uses try/except to catch invalid input
- Shows a clear error message when the input is not a number

# How to use
Run the system and type any number, including decimals.
If the input is valid, the program confirms it.
If the input is invalid, it warns the user.
Type "stop" to exit the system.
